# MiP_ProtoPack
A prototyping daughterboard to @adamgreen's [MiP_ProMini-Pack](https://github.com/adamgreen/MiP_ProMini-Pack).  This board allows additional sensors and capability to be added to the MiP_ProMiniPack.
